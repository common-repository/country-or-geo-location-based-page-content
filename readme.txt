== Description ==
This plugin provides the capability to display different content in different country based in the ID given to the element
Ex: id="US" id="GB" to show content in US and UK respectively
Currently we support 4 countries, United States, United Kingdom, India, Germany
Put respective country codes as id's for an elemnt to show in that particular country (US, GB, IN, DE)

Note : This Plugin doesn't support inside widgets

== Installation ==
Download and activate from the Plugins page of your WordPress site.
